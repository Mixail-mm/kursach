# kursach

# 1. Установить CMake и MinGW, добавить пути к папкам ./bin в PATH системных переменных
# 2. Разместить файлы с вариантом "variant_*" в одну директорию с программами. * необходимо заменить на ваш вариант, также необходимо изменить путь в самом коде программы.
# 3. Запустить файл build_and_run.bat, программы автоматически собируться и выполнятся в необходимом порядке. Выходные данные сохранятся в ./output
